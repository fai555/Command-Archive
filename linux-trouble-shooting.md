# Linux Trouble Shooting
List all open ports in current system
```bash
ss -tunl
```

# List available disk space in the file system
```bash
df -h
df -hi
df -a #show files with zero bytes
df -h /path/to/folder #show disk space in a specific folder
```

# List available
